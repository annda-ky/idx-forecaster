## üöÄ Getting Started

### Prerequisites

- **Node.js** 18+
- **Python** 3.10+
- **Supabase** Project

### 1. Clone & Install

```bash
git clone https://github.com/yourusername/idx-forecaster.git
cd idx-forecaster
```

### 2. Frontend Setup

```bash
cd apps/web
npm install
# Create .env.local with Supabase credentials
npm run dev
```

### 3. Worker Setup (AI Engine)

```bash
cd apps/worker
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
